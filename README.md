# Overview

A sales application using Java and Spring Boot on the backend and Ionic on the frontend.

## Technologies

Here are the technologies used in this project.

* Java 11
* Spring Boot
* PostgreSQL
* Ionic
* Angular

## Services Used

* Github
* Heroku
* Amazon S3
